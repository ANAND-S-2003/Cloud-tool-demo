# TaskFlow Workspace

A lightweight task management website built with plain HTML, CSS, and JavaScript.

## Features

- Create, edit, complete, and delete tasks
- Search by title or description
- Filter by status and priority
- Sort by due date, priority, status, or newest
- Dashboard metrics and a focus card for the highest-priority active task
- `localStorage` persistence so tasks stay available in the same browser
